# Password-g-n-rator


Un script Python pour générer des mots de passe sécurisés.

Utilisation
Pour utiliser le générateur de mots de passe :
```bash
python password_gen.py          # génère un mot de passe de 12 caractères (avec symboles)
python password_gen.py -l 20    # génère un mot de passe de 20 caractères
python password_gen.py -s       # génère un mot de passe sans symboles

Pré-requis
- Python 3.x doit être installé.
- Aucune dépendance externe requise (utilise les modules standards de Python).

Fonctionnalités
- Génère des mots de passe robustes avec lettres, chiffres et symboles (par défaut).
- Options pour personnaliser la longueur et exclure les symboles.

Pourquoi ça ?
- **Clarifie l’usage** : les utilisateurs savent comment lancer le script et avec quelles options.
- **Indique les pré-requis** : utile si quelqu’un veut tester le script.
- *Simple et efficace* : pas besoin de détails techniques super poussés pour un petit outil comme ça
